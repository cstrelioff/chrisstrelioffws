Hi, I'm Christopher Strelioff and you've found my home on the internet. All my friends call me Chris -- you should too. This page has links to some of my projects, ranging from html-css-javascript experiments and machine learning examples using python to my publications on Bayesian inference and disease dynamics.

I am very interested in math, science, art and programming as tools to do creative things and would love to help any project(s) that use or promote these ideas. If you are in the (East) Bay Area and are working on teaching (or using) python or javascript in a setting open to all types of people I would probably like to help out -- contact me!

I also have a passion for the arts -- music, film and more. I use my lastfm account (link above) to keep track of my listening habits and find new music. Contact with me there if you'd like to talk music -- all types! I also like to make music/sound/noise -- using ableton live, cycling 74's max, moog's werkstatt as well as other tools. If you are interested in this type of thing, or, using python and javascript to analyze and produce music (web audio api maybe??) please touch base -- email above.
